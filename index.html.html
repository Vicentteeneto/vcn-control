Dispositivo
Essas opções ficam disponíveis apenas para administrador. Usuários comuns não veem nem acessam esta tela.
Modo manutenção bloqueia comandos comuns no app. Para bloqueio físico total, o ESP32 também precisa ler essa configuração.
Wi-Fi do ESP32
Quando o ESP32 ainda não está na internet, o app não consegue configurá-lo pelo Supabase. O modo correto é o ESP32 criar uma rede local de configuração.
Para trocar o Wi-Fi: o ESP32 deve entrar no modo configuração e criar a rede VCN-Control-Setup. Depois conecte o celular nessa rede e acesse 192.168.4.1.
Esses botões enviam comandos administrativos. Eles só terão efeito quando o firmware do ESP32 estiver preparado para as ações config_wifi e reiniciar_esp32.
